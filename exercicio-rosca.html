Rosca Direta

A rosca direta é um exercício clássico para fortalecer os músculos bíceps braquial.

Execução Correta

  • Segure a barra com as palmas voltadas para cima;
  • Mantenha os cotovelos junto ao corpo;
  • Flexione os braços até a altura do peito;
  • Desça o peso de forma controlada.

Dicas

✔ Evite balançar o corpo durante o movimento.
✔ Controle a descida para maior ativação muscular.
✔ Não use cargas que comprometam a técnica.
